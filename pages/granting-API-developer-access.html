Overview

This document outlines the procedure for granting an API developer access to Apigee.

Prerequisites

Before granting access to a user, ensure that the TeamDynamix request has been processed and approved by the API Directory / Service Management team.

Step-by-Step Procedure

Once an API developer access request has been approved, the developer should be added to the apidir-developers@umich.edu MCommunity group. Membership to this group will grant the user access to the Apigee Console, where they will be able to develop and test new API proxies.

Developers should consult the API Developer Training Materials for more information about Apigee X and API development.

Please contact apidir-contact@umich.edu if you have any questions.
